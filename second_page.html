HTML

HTML (HyperText Markup Language) - это язык разметки, используемый для создания структурированных документов в сети Интернет.

  • HTML отвечает за описание структуры и содержимого веб-страницы.
  • Позволяет определять заголовки, параграфы, списки, ссылки и другие элементы для управления контентом на веб-странице.
  • Используется для создания статических веб-страниц или как основа для динамического контента.
        
    <h1>Заголовок первого уровня</h1>
    <p>Это параграф с текстом.</p>

CSS

CSS (Cascading Style Sheets) - это язык таблиц стилей, используемый для оформления веб-страниц.

  • CSS отвечает за задание внешнего вида элементов на веб-странице.
  • Позволяет устанавливать цвет текста, размер шрифта, расстояния между элементами и другие аспекты оформления страницы.
  • Применяется для добавления стилей к HTML-элементам.
            
    body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    }
    h1 {
    color: blue;
    text-align:center;
    }

Javascript

JavaScript - это язык программирования, который позволяет добавлять интерактивность на веб-странице.

  • JavaScript отвечает за обработку событий и изменение содержимого страницы без необходимости перезагрузки всей страницы.
  • Позволяет создавать сложные функциональные возможности на сайте, такие как формы обратной связи или анимации интерфейсов пользователя.
  • Используется как основной язык программирования для клиентской части web-приложений.
            
    function changeColor() {
    document.getElementById("myElement").style.color = "red";
                  }
